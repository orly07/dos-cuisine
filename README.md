Dos Cuisine — Restaurant Website

Overview
Dos Cuisine is a responsive restaurant website designed to showcase the restaurant's offerings, story, and contact information. It features a clean, modern dark theme with gold accents to match the brand’s aesthetic. The website has three main pages: Home, About Us, and Contact Us.

Design Choices
1. Dark Theme with Gold Accent
   - Background color: #050608 for a cozy, modern, and elegant restaurant feel.
   - Gold color #cf9749 used for buttons, hover states, and highlights to create brand identity and visual contrast.

2. Responsive Layout
   - Used Bootstrap 5 grid system to make the website fully responsive for desktop, tablet, and mobile.
   - Hero section has separate images for desktop and mobile to optimize visual appeal.

3. Typography
   - Use Poppins for a modern sans-serif font throughout the website for clean, readable, and approachable text.
   - Headings larger for emphasis and hierarchy, body text for comfortable reading.

4. Card-Based Layout for Featured Dishes
   - Cards have subtle shadow on hover to indicate interactivity.
   - Image containers maintain consistent height for uniformity.

5. Navigation
   - Fixed-top navbar for easy access to all pages.
   - Hover color and active state matches brand color for consistency.

6. UI/UX
   - Buttons with smooth hover transitions to improve interactivity.
   - Clear sections with padding/margins for readability and breathing space.
   - Contact form included for user convenience (UI only).

**Frameworks / Libraries Used**

1. **Bootstrap 5**
   - For responsive grid system, navbar, buttons, and spacing utilities.

3. **Bootstrap Icons**
   - For star ratings, contact icons, and visual elements.

4. **Custom CSS**
   - All additional styling is written in /assets/css/ for page-specific layouts.

Notes

- All assets are included (images, CSS, fonts).
- Buttons and cards include hover states for better interactivity.
- The website is fully responsive using Bootstrap’s grid system and custom media queries.
- No backend functionality included; contact form is UI-only.
